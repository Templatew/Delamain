# Delamain
Trackmania NF RL C++


**Uses TMI made by https://github.com/donadigo**

* Install TMNF
* Download and unpack in game folder: https://tomashu.pages.dev/modloader/modloader/TMLoader-1.0.1-win32.zip
* Download and install TMInterface 1.4.3: https://donadigo.com/files/TMInterface/TMInterface_1.4.3_Setup.exe
* change ModLoader.ini to activate TMI
* Launch game
* Launch program TMInterface.py
